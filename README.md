# نموذج التحليل المالي - برنامج نمو التطبيقي 📊

هذا المستودع يحتوي على الأتمتة البرمجية للحقيبة التدريبية الخاصة بـ **المؤتمر السعودي للمحاسبين والماليين والقانونيين (SCAFI)**، والمقدمة عبر **أكاديميتي (My Academy)**.

## 🎯 أهداف النموذج
- الفصل بين المدخلات المالية والحسابات التلقائية.
- مقارنة الأداء المالي وجوانب القوة والضعف بين شركتي `stc` و `موبايلي` لعام 2023.
- حساب نسب الربحية (ROE, ROA)، السيولة (نسبة التداول)، المديونية (D/E)، ومضاعفات التقييم (P/E, P/B).

## 🚀 التشغيل

pip install -r requirements.txt
python main.py
